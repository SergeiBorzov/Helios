Helios
=========
CPU renderer I work on in my spare time to improve my understanding of rendering techniques and algorithms.

- [ ] Point light

- [ ] Refraction BTDF

- [ ] Cook-Torrance BRDF
### Bump mapping
- [ ] Bump mapping

- [ ] Denoise?
### Antialiasing
- [ ] Supersampling
### Plugin
- [ ] Blender plugin
